Project Name: Houseek
Description:
Houseek is a web application that allows users to rent or buy apartments directly without intermediaries. Users can browse apartment listings, like apartments, message apartment owners, and upload apartment advertisements. Each user can upload up to 5 apartment ads for free; after that, they must make a payment via Stripe to continue posting. The backend is built with Spring Boot, JPA/Hibernate, and MySQL, with JWT for authentication and Firebase for storing images.

Payment Integration: After 5 free uploads, users must pay via Stripe to post more apartments.

Image Uploads: Upload apartment and user profile images using Firebase Storage.

Admin Panel: Manage apartments, users, and transactions (for admins).

Search & Filter: Find apartments based on location, price, and type.
